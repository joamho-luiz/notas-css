## Arquitetura do Arquivo

Usar edentação para definir quanto aquelas propriedades são soberanas.

Abusar de comentários.

* **CONTEÚDO** /* Como vai ser o conteúdo */  

    * Texto (Cor, Sombreamento, Riscos , Alinhamento, Espaçamento, Transformações)
    * Fonte (Família, Tamanho, Peso, Estilo)
    * Mais propriedades relacionadas ao conteúdo (Em ordem alfabética.)

* **CAIXA** /* Como vai ser a caixa */    

    * Width e Height (Overflow)
    * Background
    * Padding
    * Border
    * Outline
    * Margin

* **POSIÇÃO** /* Onde vai ficar a caixa */

    * Display
    * Position
    * Top, Right, Bottom, Left
    * Float
    * Clear

* **OUTRAS/ESPECIFICAS** /* Propriedades exclusiva do elemento ou que não relacionam a divisões anteriores. Ordenar em ordem alfabética */
